# Trabalho Prático - Engenharia de Dados

Projeto desenvolvido para a disciplina de Engenharia de Dados (UFS), 
sob orientação do Prof. André Britto de Carvalho.

## Estrutura

- `crud_postgres.py` — CRUD relacional (PostgreSQL, hospedado em RDS/AWS)
- `crud_mongo.py` — CRUD NoSQL (MongoDB, hospedado em EC2/AWS)
- `validacoes_mongo.py` — Schema Validation e índices únicos do MongoDB
- `testes_postegres.txt` / `testes_mongo.txt` / `testes_validacoes.txt` — 
  blocos de teste usados para validar cada método
